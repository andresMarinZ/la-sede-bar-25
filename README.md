# La Sede Bar 25 🍺

Sitio web corporativo de La Sede Bar 25, desarrollado en Angular con SCSS.

## ✨ Características
- Sitio informativo del negocio
- Integración con la API pública de [football-data.org](https://www.football-data.org/) 
  para mostrar información en tiempo real del mundial
